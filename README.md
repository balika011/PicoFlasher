# PicoFlasher
Open source XBOX 360 NAND flasher firmware based on Raspberry Pi Pico

## Wiring:

| Pico | Xbox |
| ------------- | ------------- |
| GP16  | SPI_MISO  |
| GP17  | SPI_SS_N |
| GP18  | SPI_CLK  |
| GP19  |  SPI_MOSI |
| GP20  |  SMC_DBG_EN |
| GP21  | SMC_RST_XDK_N  |
| GND  |  GND |
